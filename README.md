📚 Java Collections: List, Set e Map

Este repositório contém exemplos práticos e explicações sobre as principais interfaces de coleções do Java. O objetivo é demonstrar o domínio sobre como organizar, armazenar e manipular dados de forma eficiente, garantindo a performance e a integridade das informações.

🚀 Estruturas Exploradas
O projeto detalha o funcionamento e a sintaxe das três estruturas fundamentais:

1. 📝 List (ArrayList)
Conceito: Uma sequência ordenada de elementos que permite duplicatas.

Uso: List<String> funcionarios = new ArrayList<>();

Destaque: Ideal para quando a ordem de inserção é importante e você precisa acessar elementos através de índices.

2. 🛡️ Set (HashSet)
Conceito: Uma coleção que não permite elementos duplicados.

Uso: Set<String> produto = new HashSet<>();

Destaque: Focado em unicidade. Utiliza algoritmos de Hash para buscas extremamente rápidas, porém não garante a ordem dos elementos.

3. 🗺️ Map (HashMap)
Conceito: Uma estrutura baseada em pares de Chave e Valor (Dicionário).

Uso: Map<Integer, String> clientes = new HashMap<>();

Destaque: As chaves são únicas e servem como indexadores rápidos para recuperar valores complexos, simulando o comportamento de chaves primárias de bancos de dados.
